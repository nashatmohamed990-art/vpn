# 🤖 بوت VPN - freevmess

بوت تيليجرام بيجيب روابط V2ray تلقائيًا من freevmess.com

---

## 📦 التثبيت على Linux Mint (لابتوبك)

افتح Terminal وانسخ الأوامر دي واحدة واحدة:

```bash
# 1. روح لمجلد البوت
cd ~/vpn_bot

# 2. ثبّت المكتبات
pip install -r requirements.txt --break-system-packages

# 3. شغّل البوت
python3 bot.py
```

---

## ▶️ تشغيل البوت

```bash
python3 bot.py
```

هتشوف: `✅ البوت شغال!`

افتح تيليجرام وابعت `/start` للبوت.

---

## 🚀 تشغيل على سيرفر مجاني (Railway.app) — 24/7

1. اعمل حساب على https://railway.app
2. ارفع الملفات (bot.py + requirements.txt)
3. البوت هيشتغل 24/7 حتى لما اللابتوب مقفول

---

## 📱 طريقة الاستخدام

1. ابعت `/start`
2. اختار البروتوكول (V2ray / Vmess WS / Vless WS / Trojan)
3. اختار البلد
4. انسخ الرابط وحطه في V2rayTun ✅

أو اضغط **⚡ تلقائي** وهيجيبلك أحسن رابط بدون ما تختار حاجة.
